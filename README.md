# gameOfThreeFrontend

Frontend for gameOfThree

## Build Setup

``` bash
# install dependencies
npm ci

# serve with hot reload at localhost:3000
npm start
```
The project is currently hosted at https://game-of-3-frontend.herokuapp.com/
